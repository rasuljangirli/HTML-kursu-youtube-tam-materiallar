# HTML Kursu – YouTube Tam Materiallar

Youtube kanalımda dərs videolarından yararlanmaq üçün [klikləyin](https://www.youtube.com/@rasul_jangirli)

Medium hesabımda məqalələrimlə tanış olmaq üçün [klikləyin](https://medium.com/@rasuljangirli)

Bu repo YouTube-da paylaşdığım **HTML Dərsləri (0-dan)** kursunun bütün dərs materiallarını və praktiki layihələrini əhatə edir. Kurs yeni başlayanlar üçün əsas anlayışlardan başlayır və daha sonra real dünya layihələrinə qədər gedir. Buradakı materiallar həm nəzəri bilikləri, həm də praktiki bacarıqları öyrənmək üçün istifadə oluna bilər. Qeyd edim ki CSS Dərslərində də biz zatən HTML praktika edirik.

---

## Kursun Məqsədi

- HTML-in əsas anlayışlarını öyrətmək (tags, attributes, element strukturu, document flow)

- Veb səhifələrin düzgün strukturun qurmağı göstərmək (doctype, head, body, meta tagları)

- Semantik HTML-i mənimsətmək və onun SEO və accessibility-ya təsirini izah etmək (header, nav, main, article, section, footer və s.)

- Formlar və input tipləri ilə işləməyi öyrətmək — validasiya və istifadəçi-dostu form strukturu yaratmaq (form, input, label, required və HTML5 validation)

- Media elementlərindən istifadə etməyi göstərmək (img, audio, video, source, iframe) və media üçün əla təcrübələr

- HTML və CSS-in birlikdə necə işlədiyini və struktur-vizual ayırmanı izah etmək (semantic structure vs presentation)

- SEO əsaslarını tətbiq etmək — title, meta description, canonical, structured data (əsas anlayış)

- Layihələr vasitəsilə praktiki bacarıqları inkişaf etdirmək və müasir veb standartlarına uyğun kod yazmaq
